# LLM Context for ngocyen99tb.lelouvincx.com

## Site Overview
This is an educational documentation website focused on data analysis and SQL learning resources.

## Main Content Areas

### 1. Olist Brazilian E-commerce Project (/docs/olist/)
A comprehensive data analysis project using real Brazilian e-commerce data from Olist.
- **Overview**: Introduction to the project, Brazil, and Olist company
- **Dataset**: Documentation of the database schema and relationships
- **SQL Missions**: Hands-on SQL exercises organized by difficulty levels
  - Level 0: Environment setup
  - Level 1: SQL Exploration (filtering, joins, aggregations, date handling)
  - Each task includes business context, goals, hints, and success criteria

### 2. LeetCode SQL50 (/docs/leetcode-sql50/)
SQL learning materials and problem-solving guides:
- **SQL Note**: Comprehensive SQL syntax reference in Vietnamese
  - SELECT, WHERE, JOIN, Aggregate functions, CASE WHEN, etc.
- **SQL 50 Problems**: Curated collection from LeetCode's top SQL problems
  - Organized by topics: SELECT, JOINS, Aggregate Functions, Sorting/Grouping, Subqueries, etc.

## Target Audience
- Junior data analysts
- SQL learners
- Students learning data analysis with real-world datasets

## Teaching Approach
- Story-driven learning (e.g., "Rio de Janeiro Crisis" narrative)
- Hands-on exercises with business context
- Progressive difficulty levels
- Hints provided without giving away solutions
- Success criteria for self-assessment

## Technical Details
- Built with: Docusaurus
- Languages: English, Vietnamese (mixed)
- Format: Markdown/MDX documentation
- Features: Interactive exercises, video embeds, collapsible hints

## How LLMs Should Use This Content
1. When helping with SQL learning, reference the structured approach used in the missions
2. For Brazilian e-commerce analysis, use the Olist dataset structure
3. Maintain the pedagogical style: provide hints rather than direct answers
4. Consider business context when explaining SQL queries
5. Use the progressive difficulty approach when teaching
